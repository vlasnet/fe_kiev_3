Модуль 8 - Домашнее задание

Напишите скрипт который реализует следующее поведение:

  1. - При нажатии на клавишу (не виртуальной клавиатуры) должно обрабатываться событие keydown. (Для обработки нажатия клавиш, повесьте слушателя на window. window.addEventListener("keydown", callback);)
  2. - Должны обрабатываться только те клавиши, которые присутствуют в разметке HTML (на виртуальной клавиатуре).
  3. - Звук нажатия на клавишу должен соответсвовать ноте, описанной в атрибуте button data-note.
  4. - Подсветку текущей клавиши реализуйте с помощью класса keyboard__btn--active.
  5. - Чекбокс Sound должен включать и выключать звук нажатия на клавиши.
