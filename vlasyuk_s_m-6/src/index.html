Домашнее задание 6

Выполнить:

Задание 1

Преобразовать всю текущую логику в один объект keyTrainer с методами, методы обращаются к полям по this.

Задание 2

Добавить функционал ввода кол-ва букв которые мы хотели бы потренировать. Добавить все соотвествующие проверки введеного пользователем значения. Теперь пользователь сам может выбирать себе кол-во символов на упражнение.

Задание 3

Создать метод обьекта nextChar(), который выводит в консоль по очереди то кол-во случайных букв из текущего алфавита, которое ввел пользователь.

Задание 4

Сделать вывод букв с помощью nextChar() через prompt, с проверкой того что введет пользователь. По окончанию упражнения посчитать и вывести количество ошибок которые он сделал.
