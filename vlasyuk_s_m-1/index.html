Домашнее задание №1

Выполнить:

  1. Создать три const - строки, содержащие символы клавиатуры верхнего, среднего и нижнего ряда английского языка. Например для верхнего ряда русской раскладки строка будет выглядеть 'йцукенгшщзхъ'.
  2. Создать три const переменные с числом, содержащим длинну строк, полученных ранее.
  3. Используя charAt, для каждой строки получить первый и последний символ для каждой из трех строк, используйте имеющиеся константы.
  4. Сделайте поиск позиции для символов [ и ] с помощью indexOf и сохраните результат в переменных.
  5. Используя шаблонные строки, вывести названия всех переменных и полученные значения.
